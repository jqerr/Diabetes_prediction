# Galaxy M — Diabetes Prediction Pipeline

A reproducible ML experiment pipeline for binary classification on the Pima Indians Diabetes dataset. Each experiment tests one idea, results are tracked automatically, and every run is tied to a git commit.

---

## Dataset

**Pima Indians Diabetes** — 768 samples, 8 features, binary target (`Outcome`).

Download from [Kaggle](https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database) and place it at:
```
data/diabetes.csv
```

---

## Setup

```bash
git clone https://github.com/<your-username>/Galaxy_M.git
cd Galaxy_M

pip install -r requirements.txt
```

---

## Reproduce the Full Pipeline

**Step 1 — EDA**
Open and run `notebook/1_eda.ipynb`

**Step 2 — Preprocessing**
Open and run `notebook/2_preprocess.ipynb`

Outputs to `data/preprocessed_data/` (gitignored). Produces:
- `X_train.csv`, `X_test.csv`, `y_train.csv`, `y_test.csv`
- `X_train_scaled.csv`, `X_test_scaled.csv`
- `scaler.pkl`

**Step 3 — Run an experiment**
```bash
python src/train.py experiments/exp_005_xgb_tuned.yaml
```

Results are appended to `results/experiment_log.csv`.

**Step 4 — Final evaluation** *(run once, at end of project)*
```bash
python src/evaluate.py experiments/exp_005_xgb_tuned.yaml
```

## Run Your Own Experiment

Create a YAML file in `experiments/`:

```yaml
experiment_id: exp_006
description: "Your description"
hypothesis:   "What you are testing"

data:
  use_scaled: false         # true for linear models / SVM / KNN

# optional — feature engineering
features:
  module: src.features
  transforms: [interactions_v1]

# optional — class imbalance oversampling
sampling:
  method: imblearn.over_sampling.SMOTE
  params:
    random_state: 42

# optional — hyperparameter search
search:
  scoring: roc_auc
  n_iter: 50
  param_grid:
    n_estimators: [100, 200, 300]
    max_depth: [3, 4, 5, 6]

model:
  type: sklearn.ensemble.RandomForestClassifier   # any dotted import path
  params:
    random_state: 42
```

Then run:
```bash
python src/train.py experiments/exp_006_your_experiment.yaml
```

## Experiment Results

All experiments use 5-fold stratified CV on the train set. Test set is sealed.

| Experiment | Model | Key Idea | ROC-AUC | F1 | Recall |
|---|---|---|---|---|---|
| exp_001 | Random Forest | Baseline | 0.9394 | 0.8123 | 0.7929 |
| exp_002 | XGBoost | XGB baseline + scale_pos_weight | 0.9502 | 0.8232 | 0.8177 |
| exp_003 | XGBoost | Pairwise interaction features | 0.9451 | 0.8217 | 0.8175 |
| exp_004 | XGBoost + SMOTE | SMOTE for class imbalance | 0.9458 | 0.8361 | 0.8468 |
| **exp_005** | **XGBoost + SMOTE + tuned** | **Hyperparameter tuning** | **0.9487** | **0.8192** | **0.8634** |

## Reproducibility

Every experiment run logs:
- Timestamp
- All model hyperparameters
- CV metrics (mean ± std)
- Git commit hash of the code that produced the result

To reproduce any experiment exactly:
```bash
git checkout <git_hash from experiment_log.csv>
python src/train.py experiments/<experiment>.yaml
```
